Welcome to WiseAgent SDK's documentation!
======================================
WiseAgent是一个专为物联网所设计的消息队列，只需通过简单几步配置即可快速使用。
WiseAgent SDK分为云端SDK与客户端SDK，云端SDK面向第三方开发者， 客户端面向普通用户。

开发者可通过云端SDK快速接入WiseAgent建立第三方网站，
普通用户可以通过客户端SDK接入此网站快速且方便的使用稳定的消息队列服务，
网站可根据自身需求自定义用户上传数据的展示形式，并提供额外服务。
